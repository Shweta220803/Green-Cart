# Grocery Shop Application

A full-stack Grocery Shop web application where users can browse groceries, add products to cart, and manage them with a simple and interactive UI.  
Built using the MERN stack with Tailwind CSS for styling and implemented Stripe payment gateway integration.

## 🚀 Features

- User-friendly browsing experience
- Add, update, and delete grocery items
- Real-time cart management
- Secure user authentication and authorization
- Responsive design for all devices
- Backend APIs for product and user management
- Smooth integration between frontend and backend

##  Technology Stack

**Frontend:**  
- React.js  
- Tailwind CSS  

**Backend:**  
- Node.js  
- Express.js  

**Database:**  
- MongoDB  
**Payment Gateway Integration:**
  - Stripe API



## 📋 Setup Instructions

1. **Clone the repository**

2. **Navigate into the project directory**

3. **Install frontend dependencies**
    npm install

4. **Install backend dependencies**

    npm install

5. **Set up environment variables**

Create a `.env` file in the `backend/` folder with:

MONGO_URI=your_mongodb_connection_string
PORT
JWT_SECRET=your_secret_key

6. **Run the application**

Open two terminals:

- One for backend:

cd backend
npm start

- One for frontend:

cd frontend
npm run dev
